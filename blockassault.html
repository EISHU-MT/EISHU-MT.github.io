BlockAssault

What is BlockAssault?

BlockAssault is a First/Third person shooter game. Focused to be the Better PVP Engine on Minetest.
This game has several weapons, Such as the:

  • AK-47
  • AWP
  • Deagle (Desert Eagle)
  • All types of grenades
  • M200
  • Many more...

    • Inspired by Minetest CTF, Minetest Coalition Warfare

    Has bot support: YES

    License: LGPL-2.1-only

    This game dont require expensive computer to run

    ~How to run the game?

    1. Install the Minetest Engine Here!
    2. In Minetest, go to Content and click on Browse online content
    3. On "Browse online content", search 'blockassault'
    4. Next, search Bots (Optional)
    5. When all finished. Go to "Start Game" tab and press the gun icon
    6. On BlockAssault tab, click "New" and make a new world, Be sure the Mapgen is 'singlenode'
    7. Then, Press 'Play Game'

    Default Keyboard Order:

    W, A, S, D: Move player
    E: Reload gun ammo (Aux1)
    I: Inventory
    Space: Jump
    Left Click Mouse: Shoot/Dig
    Right Click Mouse: Aim/Reload gun (Not Ammo)
    More keys order are in Minetest Settings
